## Activación de Entorno de Desarrollo y Ejecución de Servidor

### Windows

```bash
cd venv/venv01/Scripts/
activate

cd projects/duoc_2020_django/

python manage.py runserver
```

### Macintosh

```bash
source venv/venv01/bin/activate

cd projects/duoc_2020_django/

python3 manage.py runserver
```
